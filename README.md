# 美股科技股每日分析報告生成器

自動分析美股科技股（AI產業、區塊鏈、CPO、低軌衛星、HBM記憶體）的漲跌情況，生成包含外資目標價與台股推薦建議的專業分析報告，並透過 Gmail 發送。

## 功能特色

- **多產業涵蓋**：AI產業龍頭、區塊鏈相關、台股連動核心、CPO共封裝光學、低軌衛星、HBM記憶體
- **即時數據**：從 Yahoo Finance 獲取最新股價數據
- **智能分析**：計算漲跌幅、成交量比等關鍵指標
- **台股推薦**：根據美股表現動態生成台股投資建議
- **外資目標價**：整合外資券商目標價資訊
- **專業報告**：生成精美的 HTML 格式分析報告
- **自動發送**：透過 Gmail SMTP 自動發送報告

## 安裝依賴

```bash
pip3 install yfinance pandas pytz
```

## 使用方式

### 手動執行

```bash
python3 tech_stock_analyzer.py
```

### 定時執行（週一至週五早上 7:30）

```bash
crontab -e
```

加入以下內容：

```
30 7 * * 1-5 python3 /home/ubuntu/tech_stock_analyzer.py
```

## 報告內容

1. **漲幅前五名與跌幅前五名**
2. **成交量異常股票**（>1.5倍平均）
