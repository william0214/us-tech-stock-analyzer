# 美股科技股分析報告系統

自動化分析美股科技股（AI 產業、區塊鏈、台股連動股）的漲跌情況，生成包含外資目標價與台股推薦建議的專業分析報告，並透過 Gmail 自動發送。

## 功能特色

- **多類別股票分析**：涵蓋 AI 產業龍頭、區塊鏈相關、台股連動核心等 19 支美股
- **專業數據指標**：計算漲跌幅、成交量比、價格變動等關鍵指標
- **台股投資建議**：根據美股表現自動生成台股供應鏈投資建議
- **外資目標價追蹤**：特別標註外資調升目標價的個股
- **精美 HTML 報告**：專業排版的 HTML 格式分析報告
- **自動郵件發送**：透過 Gmail SMTP 自動發送報告

## 系統需求

- Python 3.11+
- 網路連線（用於獲取 Yahoo Finance 數據）
- Gmail 帳號與應用程式密碼

## 安裝依賴套件

```bash
pip3 install yfinance pandas pytz
```

## 使用方式

### 手動執行

```bash
python3 tech_stock_analyzer.py
```

### 定時執行（建議）

使用 cron 設定每週一到週五早上 7:30 自動執行：

```bash
# 編輯 crontab
crontab -e

# 加入以下設定（台北時間早上 7:30）
30 7 * * 1-5 /usr/bin/python3 /home/ubuntu/tech_stock_analyzer.py
```

## Gmail 設定

1. 登入 Google 帳號
2. 前往「管理您的 Google 帳戶」→「安全性」
3. 啟用「兩步驟驗證」
4. 在「應用程式密碼」中產生新密碼
5. 將密碼更新到腳本中的 `GMAIL_CONFIG`

## 執行流程

1. **獲取股票數據**：從 Yahoo Finance API 獲取 19 支美股數據
2. **計算指標**：計算漲跌幅、成交量比等關鍵指標
3. **生成台股建議**：根據美股表現自動生成台股推薦
4. **生成 HTML 報告**：產生專業排版的分析報告
5. **發送郵件**：透過 Gmail SMTP 自動發送

## 錯誤處理

- 若某支股票數據獲取失敗，會跳過並繼續處理其他股票
- 確保至少獲取到大部分股票數據後再發送報告
- 自動檢查 Gmail 連線與認證狀態

## 注意事項

- 確保網路連線正常
- Yahoo Finance API 偶爾會有延遲或錯誤
- Gmail 應用程式密碼需保持有效
- 腳本執行時間約 1-2 分鐘
- 避開週末與美股休市日執行
